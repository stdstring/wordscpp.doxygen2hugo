## PdfLoadOptions class


Allows to specify additional options when loading Pdf document into a Document object. 

## Methods

| Method | Description |
| --- | --- |
|  [PdfLoadOptions](./pdfloadoptions/)() |  |
| int32_t [get_PageCount](./get_pagecount/)() const | Gets the number of pages to read. Default is MaxValue which means all pages of the document will be read.  |
| int32_t [get_PageIndex](./get_pageindex/)() const | Gets the 0-based index of the first page to read. Default is 0.  |
| bool [get_SkipPdfImages](./get_skippdfimages/)() const | Gets the flag indicating whether images must be skipped while loading PDF document. Default is False.  |
| void [set_PageCount](./set_pagecount/)(int32_t) | Sets the number of pages to read. Default is MaxValue which means all pages of the document will be read.  |
| void [set_PageIndex](./set_pageindex/)(int32_t) | Sets the 0-based index of the first page to read. Default is 0.  |
| void [set_SkipPdfImages](./set_skippdfimages/)(bool) | Sets the flag indicating whether images must be skipped while loading PDF document. Default is False.  |
