The **Aspose.Words.Loading** namespace provides classes and enumerations that allow to specify additional options when loading documents. 

## Classes

| Class | Description |
| --- | --- |
| [Aspose::Words::Loading::ChmLoadOptions](./chmloadoptions/) | Allows to specify additional options when loading CHM document into a Document object.  |
| [Aspose::Words::Loading::DocumentLoadingArgs](./documentloadingargs/) | An argument passed into Notify().  |
| [Aspose::Words::Loading::HtmlLoadOptions](./htmlloadoptions/) | Allows to specify additional options when loading HTML document into a Document object.  |
| [Aspose::Words::Loading::LanguagePreferences](./languagepreferences/) | Allows to set up language preferences.  |
| [Aspose::Words::Loading::LoadOptions](./loadoptions/) | Allows to specify additional options (such as password or base URI) when loading a document into a Document object.  |
| [Aspose::Words::Loading::PdfLoadOptions](./pdfloadoptions/) | Allows to specify additional options when loading Pdf document into a Document object.  |
| [Aspose::Words::Loading::ResourceLoadingArgs](./resourceloadingargs/) | Provides data for the ResourceLoading() method.  |
| [Aspose::Words::Loading::RtfLoadOptions](./rtfloadoptions/) | Allows to specify additional options when loading Rtf document into a Document object.  |
| [Aspose::Words::Loading::TxtLoadOptions](./txtloadoptions/) | Allows to specify additional options when loading Text document into a Document object.  |
## Interfaces

| Interface | Description |
