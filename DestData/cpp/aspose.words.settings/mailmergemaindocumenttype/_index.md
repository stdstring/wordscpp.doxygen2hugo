## MailMergeMainDocumentType enum


Specifies the possible types for a mail merge source document.

### Values

| Name | Value | Description |
| --- | --- | --- |
| NotAMergeDocument | `0` | This document is not a mail merge document. |
| FormLetters | `0` | Specifies that the mail merge source document is of the form letter type. |
| MailingLabels | `0` | Specifies that the mail merge source document is of the mailing label type. |
| Envelopes | `0` | Specifies that the mail merge source document is of the envelope type. |
| Catalog | `0` | Specifies that the mail merge source document is of the catalog type. |
| Email | `0` | Specifies that the mail merge source document is of the e-mail message type. |
| Fax | `0` | Specifies that the mail merge source document is of the fax type. |
| Default | `0` | Equals to **NotAMergeDocument** |
