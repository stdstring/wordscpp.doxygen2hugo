## TextColumnCollection class


A collection of TextColumn objects that represent all the columns of text in a section of a document. 

## Methods

| Method | Description |
| --- | --- |
| int32_t [get_Count](./get_count/)() | Gets the number of columns in the section of a document.  |
| bool [get_EvenlySpaced](./get_evenlyspaced/)() | **True** if text columns are of equal width and evenly spaced.  |
| bool [get_LineBetween](./get_linebetween/)() | When **true**, adds a vertical line between columns.  |
| double [get_Spacing](./get_spacing/)() | When columns are evenly spaced, gets or sets the amount of space between each column in points.  |
| double [get_Width](./get_width/)() | When columns are evenly spaced, gets the width of the columns.  |
| System::SharedPtr< Aspose::Words::TextColumn > [idx_get](./idx_get/)(int32_t) | Returns a text column at the specified index.  |
| void [set_EvenlySpaced](./set_evenlyspaced/)(bool) | Setter for Aspose::Words::TextColumnCollection::get_EvenlySpaced.  |
| void [set_LineBetween](./set_linebetween/)(bool) | Setter for Aspose::Words::TextColumnCollection::get_LineBetween.  |
| void [set_Spacing](./set_spacing/)(double) | Setter for Aspose::Words::TextColumnCollection::get_Spacing.  |
| void [SetCount](./setcount/)(int32_t) | Arranges text into the specified number of text columns.  |
