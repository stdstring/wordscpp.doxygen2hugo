## ChapterPageSeparator enum


Defines the separator character that appears between the chapter and page number.

### Values

| Name | Value | Description |
| --- | --- | --- |
| Hyphen | `0` | A colon. |
| Period | `0` | A period. |
| Colon | `0` | A colon. |
| EmDash | `0` | An emphasized dash. |
| EnDash | `0` | A standard dash. |
