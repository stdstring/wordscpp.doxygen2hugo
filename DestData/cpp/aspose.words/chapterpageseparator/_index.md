## ChapterPageSeparator enum


Defines the separator character that appears between the chapter and page number.

### Values

| Name | Value | Description |
| --- | --- | --- |
| Hyphen | 0 | A colon. |
| Period | 1 | A period. |
| Colon | 2 | A colon. |
| EmDash | 3 | An emphasized dash. |
| EnDash | 4 | A standard dash. |
