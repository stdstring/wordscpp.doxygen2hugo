## Range class


Represents a contiguous area in a document.

| [get_Fields](./get_fields/)() | Returns a Fields collection that represents all fields in the range.  |
| [get_FormFields](./get_formfields/)() | Returns a FormFields collection that represents all form fields in the range.  |
| [get_StructuredDocumentTags](./get_structureddocumenttags/)() | Returns a StructuredDocumentTags collection that represents all structured document tags in the range.  |
