## TabAlignment enum


Specifies the alignment/type of a tab stop.

### Values

| Name | Value | Description |
| --- | --- | --- |
| Left | `0` | Left-aligns the text after the tab stop. |
| Center | `0` | Centers the text around the tab stop. |
| Right | `0` | Right-aligns the text at the tab stop. |
| Decimal | `0` | Aligns the text at the decimal dot. |
| Bar | `0` | Draws a vertical bar at the tab stop position. |
| List | `0` | The tab is a delimiter between the number/bullet and text in a list item. |
| Clear | `0` | Clears any tab stop in this position. |
