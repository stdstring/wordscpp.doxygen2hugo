## WarningSource enum


Specifies the module that produces a warning during document loading or saving.

### Values

| Name | Value | Description |
| --- | --- | --- |
| Unknown | `0` | The warning source is not specified. |
| Layout | `0` | Module that builds a document layout. |
| DrawingML | `0` | Module that renders DrawingML shapes. |
| OfficeMath | `0` | Module that renders OfficeMath. |
| Shapes | `0` | Module that renders ordinary shapes. |
| Metafile | `0` | Module that renders metafiles. |
| Xps | `0` | Module that renders XPS. |
| Pdf | `0` | Module that renders PDF. |
| Image | `0` | Module that renders images. |
| Docx | `0` | Module that reads/writes DOCX files. |
| Doc | `0` | Module that reads/writes binary DOC files. |
| Text | `0` | Module that reads/writes plaintext files. |
| Rtf | `0` | Module that reads/writes RTF files. |
| WordML | `0` | Module that reads/writes WML files. |
| Nrx | `0` | Common modules that are shared between DOCX/WML reader/writer modules. |
| Odt | `0` | Module that reads/writes ODT files. |
| Html | `0` | Module that reads/writes HTML/MHTML files. |
| Validator | `0` | Module that verifies model consistency and validity. |
| Xaml | `0` | Module that reads/writes Xaml files. |
| Svm | `0` | Module that reads Svm files. |
| MathML | `0` | Module that reads W3C MathML files. |
| Font | `0` | Module that reads font files. |
| Svg | `0` | Module that reads SVG files. |
| Markdown | `0` | Module that reads/writes Markdown files. |
| Chm | `0` | Module that reads CHM files. |
| Epub | `0` | Module that reads/writes EPUB files. |
