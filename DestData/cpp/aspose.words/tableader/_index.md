## TabLeader enum


Specifies the type of the leader line displayed under the tab character.

### Values

| Name | Value | Description |
| --- | --- | --- |
| None | `0` | No leader line is displayed. |
| Dots | `0` | The leader line is made up from dots. |
| Dashes | `0` | The leader line is made up from dashes. |
| Line | `0` | The leader line is a single line. |
| Heavy | `0` | The leader line is a single thick line. |
| MiddleDot | `0` | The leader line is made up from middle-dots. |
